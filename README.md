# OpenMIPS
这是一个根据雷思磊老师的《自己动手写CPU》的实践项目，初衷是为了学习Verilog编写CPU的流程，熟悉硬件开发过程，并最终编写出我的第一个CPU

此CPU基于MPIS指令集，不过我打算在实现完这个CPU之后，将此CPU重写为RISC-V指令集的CPU
